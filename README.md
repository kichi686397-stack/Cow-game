## 🌟 游戏介绍

小牛桌宠是一款轻量级网页养成游戏，主角是一只可爱的小牛。你可以喂食、遛牛、钓鱼、玩小游戏，还能带着小牛穿越平行宇宙！游戏完全免费，无广告，数据保存在本地，随时继续。

---

## 🎮 玩法一览

### 🍎 喂食养牛
- 点击底部食物按钮喂食，每次获得 XP
- 累积 XP 可升级，共 **10 个等级**：小牛犊 → 小牛 → 壮牛 → 大牛 → 牛王 → 神牛 → 传奇牛 → 史诗牛 → 远古牛 → 牛神
- 吃多汁水果（西瓜/草莓/苹果等）额外补充水分

### 👆 互动
- **点击小牛** — 摸头，小牛心情好会有反应
- **长按拖动** — 遛牛散步
- **45秒不互动** — 小牛会自己玩：踢石头、追尾巴、望天发呆……

### 💧 喝水系统
- 点击左侧水槽让小牛去喝水
- 口渴值过低小牛会提醒你
- 下雨天自动缓慢补充水分

### 🎣 钓鱼
- 点击菜单 🎣 进入全屏湖边钓鱼场景
- 等浮漂沉下去 → 点击收竿 → 获得战利品和 XP
- 不同平行宇宙有不同的钓鱼池和专属物品

### 🎮 接水果小游戏
- 30 秒内左右移动篮子接住掉落的食物
- 有害物品（水母/病毒球/陨石等）接到会扣分
- 不同平行宇宙有不同的掉落物

### 🌍 平行宇宙
穿越到 5 个不同宇宙，背景、小牛颜色、访客、小游戏内容全都会变：

| 宇宙 | 风格 |
|------|------|
| 🌿 普通农场 | 默认田园风 |
| 🌃 赛博朋克 | 霓虹电路，小牛变紫 |
| 🌊 深海水下 | 气泡珊瑚，小牛变蓝绿 |
| 🚀 太空站 | 星尘飘浮，银河背景 |
| 🏯 古代农村 | 暖橙天空，复古风格 |

### 🎨 皮肤与装备
喂食或钓鱼达到条件解锁装备，给小牛戴上：

花环 🌸 · 墨镜 🕶️ · 围巾 🧣 · 大礼帽 🎩 · 圣诞帽 🎅 · 彩虹角 🌈 · 海草圈 🌿 · 船锚链 ⚓ · 贝壳冠 🐚 · 钓鱼勋章 🎖️ · 厨师帽 👨‍🍳 · 王冠 👑

### 🏆 成就系统
完成各种挑战解锁成就徽章，涵盖喂食、钓鱼、探索宇宙、访客互动等多个维度。

### 🐾 访客系统
小动物们会随机来拜访：猫咪 🐱 · 兔子 🐇 · 蝴蝶 🦋 · 小鸟 🐦 · 青蛙 🐸

## 🏅 全球排行榜

点击菜单 🏅 查看全球排名，四个维度：

- 🐄 **等级榜** — 总喂食 XP
- 🏆 **成就榜** — 解锁成就数量
- 🍎 **接水果榜** — 单局最高分
- 🎣 **钓鱼榜** — 累计钓鱼总数

点击「上传我的数据」填写名字即可上榜，排名按设备锁定，不可重复刷榜。

## 💾 数据说明

- 游戏数据保存在浏览器 **localStorage**，刷新不丢失
- 更换浏览器或设备需重新开始
- 排行榜数据存储在云端，所有玩家共享
