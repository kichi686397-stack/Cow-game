## 🌟 游戏介绍

小牛桌宠是一款轻量级网页养成游戏，主角是一只可爱的小牛。你可以喂食、遛牛、钓鱼、玩小游戏，还能带着小牛穿越平行宇宙！游戏完全免费，无广告，数据保存在本地，随时继续。

---

## 🎮 玩法一览

### 🍎 喂食养牛
- 点击底部食物按钮喂食，每次获得 XP
- 累积 XP 可升级，共 **10 个等级**：小牛犊 → 小牛 → 壮牛 → 大牛 → 牛王 → 神牛 → 传奇牛 → 史诗牛 → 远古牛 → 牛神
- 吃多汁水果（西瓜/草莓/苹果等）额外补充水分

### 👆 互动
- **点击小牛** — 摸头，小牛心情好会有反应
- **长按拖动** — 遛牛散步
- **45秒不互动** — 小牛会自己玩：踢石头、追尾巴、望天发呆……

### 💧 喝水系统
- 点击左侧水槽让小牛去喝水
- 口渴值过低小牛会提醒你
- 下雨天自动缓慢补充水分

### 🎣 钓鱼
- 点击菜单 🎣 进入全屏湖边钓鱼场景
- 等浮漂沉下去 → 点击收竿 → 获得战利品和 XP
- 不同平行宇宙有不同的钓鱼池和专属物品

### 🎮 接水果小游戏
- 30 秒内左右移动篮子接住掉落的食物
- 有害物品（水母/病毒球/陨石等）接到会扣分
- 不同平行宇宙有不同的掉落物

### 🌍 平行宇宙
穿越到 5 个不同宇宙，背景、小牛颜色、访客、小游戏内容全都会变：

| 宇宙 | 风格 |
|------|------|
| 🌿 普通农场 | 默认田园风 |
| 🌃 赛博朋克 | 霓虹电路，小牛变紫 |
| 🌊 深海水下 | 气泡珊瑚，小牛变蓝绿 |
| 🚀 太空站 | 星尘飘浮，银河背景 |
| 🏯 古代农村 | 暖橙天空，复古风格 |

### 🎨 皮肤与装备
喂食或钓鱼达到条件解锁装备，给小牛戴上：

花环 🌸 · 墨镜 🕶️ · 围巾 🧣 · 大礼帽 🎩 · 圣诞帽 🎅 · 彩虹角 🌈 · 海草圈 🌿 · 船锚链 ⚓ · 贝壳冠 🐚 · 钓鱼勋章 🎖️ · 厨师帽 👨‍🍳 · 王冠 👑

### 🏆 成就系统
完成各种挑战解锁成就徽章，涵盖喂食、钓鱼、探索宇宙、访客互动等多个维度。

### 🐾 访客系统
小动物们会随机来拜访：猫咪 🐱 · 兔子 🐇 · 蝴蝶 🦋 · 小鸟 🐦 · 青蛙 🐸

## 🏅 全球排行榜

点击菜单 🏅 查看全球排名，四个维度：

- 🐄 **等级榜** — 总喂食 XP
- 🏆 **成就榜** — 解锁成就数量
- 🍎 **接水果榜** — 单局最高分
- 🎣 **钓鱼榜** — 累计钓鱼总数

点击「上传我的数据」填写名字即可上榜，排名按设备锁定，不可重复刷榜。

## 🧭 开发与发版原则

当前版本：**v10.3.7**

当前主文件：`cowgame_deploy/index.html`

当前版本日志：`小牛桌宠_版本日志.md`

每次改动必须同步完成：

1. 更新 `cowgame_deploy/index.html` 的 `<title>` 版本号
2. 更新 `CURRENT_VER` 和游戏内 `CHANGELOG` 最新条目
3. 更新 `小牛桌宠_版本日志.md` 的对应版本条目
4. 版本号遵循 SemVer：
   - Major：新增独立核心系统 / 架构级重构
   - Minor：向下兼容的新功能
   - Patch：Bug 修复 / 小改动 / 文档与维护规则调整

技术备忘：

- 存档使用 `localStorage`，key = `pixelcow_save_v1`
- 现有小牛与访客仍包含 Canvas 绘制逻辑，重构期保持兼容
- 后续新增或重做角色动画时，优先使用图片帧 / 精灵帧，不再新增整头牛 Canvas 手绘逻辑
- Canvas 主要保留给场景特效、小游戏画布和过渡期兼容层
- 修改现有绘制函数后，需要做语法检查，避免残留代码悬挂

## 🛠️ 技术栈

- 纯原生 HTML + CSS + JavaScript，无框架
- Canvas 绘图（现有兼容层、场景特效与小游戏）
- 图片帧 / 精灵帧（后续角色动画重构方向）
- localStorage 本地存档
- Supabase 排行榜数据库（当前正式路线）
- Cloudflare Pages / Workers 静态资源部署
